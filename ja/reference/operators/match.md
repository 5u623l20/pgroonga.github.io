# `%%`演算子

`%%`演算子を使うと1つキーワードで全文検索できます。

```sql
SELECT * FROM memos WHERE content %% '全文検索';
--  id |                      content
-- ----+---------------------------------------------------
--   2 | Groongaは日本語対応の高速な全文検索エンジンです。
-- (1 行)
```

複数のキーワードで検索したいときやAND/ORを使った検索をしたいときは[`@@`演算子](query.html)を使います。
