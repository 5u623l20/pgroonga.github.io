# `pgroonga.log_path` parameter

## Description

Change output log file path. This parameter is only effective if log_type is ``file``.

## Syntax

```sql
set pgroonga.log_path = path
```

`path` is log file path.Default value is ``$PGDATA/pgroonga.log`.
